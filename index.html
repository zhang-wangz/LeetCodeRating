LeetCodeRating｜一款显示题目对应周赛难度分的浏览器插件。

在使用脚本之前，你需要为浏览器安装脚本管理器插件。

经过测试，我们确定兼容以下浏览器和插件用于安装此脚本。

:globe_with_meridians: 浏览器支持（任选其一即可）

  • Chrome 或 基于 Chromium 内核的浏览器 (Edge、360浏览器、CentBrowser等) *推荐
  • Firefox *功能兼容，但并非最佳性能

:see_no_evil: 浏览器需要安装的脚本管理器插件

  • Tampermonkey (新油猴) *插件安装链接

:page_facing_up: 安装脚本 (求star ヾ(≧▽≦*)o)

  • 从 脚本猫 安装 (提供没有科学上网伙伴下载) 通过 脚本猫 安装脚本: 点我

  • 从 GreasyFork 安装 （被中国gfw封锁，只能科学上网进行下载安装） 通过 GreasyFork 安装脚本: 点我

:test_tube: 支持的操作系统

  • Windows、MacOS、Linux 等支持 Chrome 或 Firefox 浏览器的所有操作系统 *移动平台如 iOS、Android 除外

:rocket: 贡献

Contributors List 非常欢迎你的加入！提一个 Issue 或者提交一个 Pull Request。
pr要求如下：

  1. 修改版本号信息:
  • 如果第三位<10，将第三位+1，如果第三位=10，将第二位+1，第三位=0;
  • 例 1.1.10 -> 1.2.0; 1.1.3 -> 1.1.4;
  • 如果改动更新过大，将被归为大版本更新，直接进入下个大版本，例 1.2.4 -> 1.3.0;
  1. 修改updatelog.txt文件，将更新描述在最上方加上一行，修改版本号（要求同上），并修改更新描述;
  2. 修改js脚本最上方@version为提交的新版本和js文件中变量名为version的value，保持一致，同时更新version.json文件说明更新的版本和更新内容变化;

LeetCodeRating遵循 Contributor Covenant 行为规范。
